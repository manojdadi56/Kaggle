## Summary
Implemented a pre-filter script `logprob_filter.py` that runs an inference pass on the `Nemotron-3-Nano-30B-A3B-BF16` base model to calculate token-level log-probabilities. It filters out already-easy examples by dropping rows where the minimum answer token logprob is > -0.05. I also configured the `train.py` kernel inside `train-logprob-filter-e008` to default to the generated `corpus_filtered.jsonl` dataset.
